## Compile with VS Code

Open the command palette

OSX: <kbd>Shift</kbd> + <kbd>Cmd</kbd> + <kbd>P</kbd>

Windows:
<kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>P</kbd>

Type:
`Run Build Task`
<br>
<br>

## Run program

In the terminal navigate to the root folder and run it <br>

```console
$ ./main
```
